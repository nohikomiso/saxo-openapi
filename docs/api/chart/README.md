# Chart カテゴリ

チャートデータ取得のためのエンドポイント群

## 概要

Chart カテゴリは、テクニカル分析やチャート表示に必要な価格履歴データを提供します。時間軸を指定して、OHLC (始値・高値・安値・終値) データを取得できます。

## 主なユースケース

- **チャート表示**: ローソク足チャート、ラインチャートの描画
- **テクニカル分析**: 移動平均、RSI、MACDなどのインジケーター計算
- **バックテスト**: 過去データを使用した取引戦略のテスト
- **価格履歴確認**: 特定期間の価格推移の確認

## モジュール一覧

### [charts](charts.md)

チャートデータを取得します。

- `GetChartData`: 指定した銘柄・時間軸のチャートデータ取得
- サポートされる時間軸: 1分足、5分足、1時間足、日足など
- OHLC (始値・高値・安値・終値) + 出来高データ

**主な用途**: チャート描画、テクニカル分析

**パラメータ**:
- `AssetType`: 資産タイプ (FxSpot, Stock など)
- `Uic`: 銘柄の一意識別子
- `Horizon`: 時間軸 (1 minute, 1 hour, 1 day など)
- `Count`: 取得するバーの数
- `FromDate` / `ToDate`: 期間指定
