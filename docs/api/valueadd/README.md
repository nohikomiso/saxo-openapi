# ValueAdd カテゴリ

付加価値サービスのためのエンドポイント群

## 概要

ValueAdd (VAS - Value Added Services) カテゴリは、基本的な取引機能を補完する付加価値サービスを提供します。価格アラート機能により、指定した条件で価格変動の通知を受け取ることができます。

## 主なユースケース

- **価格アラート設定**: 目標価格や損切り価格のアラート設定
- **通知管理**: アラートのON/OFF、条件変更
- **アラート履歴**: 発動したアラートの履歴確認
- **自動監視**: 24時間365日の価格監視

## モジュール一覧

### [pricealerts](pricealerts.md)

価格アラートの定義と管理を行います。

- `GetAllAlerts`: すべての価格アラート定義の取得
- `CreatePriceAlert`: 新しい価格アラートの作成
- `UpdatePriceAlert`: 既存のアラート設定の更新
- `DeletePriceAlert`: 価格アラートの削除
- `GetUserNotificationSettings`: ユーザー通知設定の取得

**主な用途**: 価格アラート設定、通知管理

**アラートの種類**:
- 価格到達アラート: 指定価格に到達したときに通知
- 変動率アラート: 指定の変動率を超えたときに通知
- 期限付きアラート: 有効期限を設定可能

**通知方法**:
- アプリ内通知
- プッシュ通知
- メール通知 (設定による)

---

## 価格アラートの仕組み

1. **アラート作成**: `CreatePriceAlert` で条件を指定してアラートを作成
2. **自動監視**: Saxo システムが24時間365日価格を監視
3. **通知発動**: 条件を満たすとユーザーに通知
4. **アラート管理**: 不要になったアラートは削除可能
